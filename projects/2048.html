Description

This project involves writing the core logic behind the popular 2048 Game.

Implementation

- Given the GUI framework of the 2048 game, write the main key pressed and game logic events

- When two of the same-numbered tiles merge, the users score increments by a factor of that tiles number

- The game is over when there are no more tiles that can be merged, and the users maximum score is outputted

Skills

- Java Programming

- Object-Oriented Design

- Event Handling and Game Mechanics
